# nvim.rc

Nvim.rc originally based on ThePrimeagen's nvim.rc.

## Manual setup

### Ruff default config

Put default ruff.toml to the following location:

- Windows: `~/AppData/Roaming/ruff/ruff.toml`
- Linux: `~/.config/ruff/ruff.toml`
